These are some online courses I've enjoyed.

From Nand to Tetris

A 2-part "learn by doing" course that guides you through the construction of a computing platform from the ground up.

Part I goes through the implementation of a simple von Neumann architecture starting with elementary logic gates and introduces machine code and assembly programming. Part II develops the software hierarchy, from the simple construction of an assembler, to the development of a virtual machine and a compiler, and finally the implementation of a basic operating system.

Part I Part II
Algorithms

An excellent introduction to algorithms and data structures.

Part I discusses the fundamentals, such as stacks and queues, merge sort and quick sort, search trees, and hash tables. Part II deals with graph algorithms, such as breadth- and depth-first search and Dijkstra's algorithm, as well as string processing algorithms, such as the famous grep algorithm.

Part I Part II
Programming Languages

A 3-part introduction to the theory of programming languages.

Part A introduces functional programming with static types through Standard ML; Part B introduces dynamic typing through Racket; and Part C introduces object-oriented programming with Ruby.

Part A Part B Part C
Machine Learning

A great overview of the field: from linear and logistic regression to neural networks, support vector machines, PCA, and clustering.

Deep Learning

A 5-course sequence on deep neural networks. The 5 courses discuss, respectively: feedforward neural networks, training techniques, structuring deep learning projects, convolutional neural networks, and recurrent neural networks.

Course
Functional Programming in Scala

The first part of a 5-course sequence; an introduction to functional programming through Scala.

I hope to take the other parts sometime. These begin with an introduction to more advanced functional concepts (monads, lazy evaluation) and continue on to parallel programming and big data analysis and conclude with a capstone project.
